# JavidTech Responsive Business Website

A modern, responsive multi-page business website developed using HTML5, CSS3, and CSS Media Queries. This project demonstrates responsive web design principles and provides a professional user experience across desktop, tablet, and mobile devices.

---

## Features

- Responsive Navigation Bar
- Multi-Page Website
- Hero Section
- About Page
- Services Section
- Project Gallery
- Contact Page
- Professional Footer
- CSS Media Queries
- Mobile Friendly Layout
- Smooth Hover Effects
- Font Awesome Icons
- Google Fonts
